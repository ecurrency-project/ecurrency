ECurrency Core integration/staging tree
=======================================

What is ECurrency?
------------------

ECurrency is an experimental digital currency that enables instant payments to
anyone, anywhere in the world. ECurrency uses peer-to-peer technology to operate
with no central authority: managing transactions and issuing money are carried
out collectively by the network. ECurrency Core is the name of open source
software which enables the use of this currency.
